The Story So Far

Aric has been retraining into coding by taking an intensive boot camp with CodeClan. Based in Edinburgh, CodeClan has been teaching him about the fundamentals of professional Software Development in Python, Javascript and Java.

Alongside learning the languages, there has been a clear focus on Test-Driven Development, as well as writing code that is both dry and human readable.

The projects listed in the portfolio are apps written during the course, some as week-long project sprints, and some as a one-day exercise.

Before The Code
